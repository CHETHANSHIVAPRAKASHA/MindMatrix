Sante Price Index 📊

Sante Price Index is an Android mobile application designed to help users track, compare, and analyze product prices in real time. The app enables consumers to make smarter purchasing decisions through price comparison, trend analysis, and price tracking features.

📱 Product Overview

Consumers often face difficulties monitoring changing product prices across multiple stores and markets. Sante Price Index provides a centralized and user-friendly solution where users can:

Search products
Compare prices
Track price trends
Save favorite products
Receive price alerts

The application is developed using Java and Kotlin in Android Studio.
